# Organizational Spending Sunburst Diagram

This repository contains a professional Sunburst Diagram visualizing organizational spending breakdown by department and expense type. Created using RAWGraphs, this 3-level hierarchical visualization (Company > Department > Expense) supports budget analysis and cost center optimization for executive presentations.

## Data Source
Generated CSV with 20 rows of realistic hierarchical data:
- Level 1 (Center): Company (total spend: ~$1,033,000)
- Level 2 (Middle Ring): Departments (e.g., Sales, Marketing, IT)
- Level 3 (Outer Ring): Expenses (e.g., Travel, Events, Salaries)
- Sized by "amount" field for proportional arcs.

Sample row: `Company,Sales,Travel,56012`

## Tools Used
- [RAWGraphs.io](https://rawgraphs.io/) for diagram creation and export.
- Chart dimensions: 512x512 PNG (publication-ready).
